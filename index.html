АНГЛИЦИЗМЫ: ДРУЗЬЯ ИЛИ ВРАГИ?

Интерактивная памятка по правильному использованию англицизмов в русском языке

Проектная работа ученика 7-Г класса Баженова Степана

ПАМЯТКА ДЛЯ ШКОЛЬНИКОВ

✅ МОЖНО ИСПОЛЬЗОВАТЬ (полезные англицизмы) ❌ ЛУЧШЕ ЗАМЕНИТЬ (есть русские аналоги)
Блогер - нет точного русского аналога Окей → Хорошо, Ладно, Согласен
Стример - новая профессия Лайк → Нравится, Понравилось
Айфон - название бренда Сторис → Истории, История в соцсети
Email - короче, чем "электронная почта" Хайп → Тренд, Популярность
Wi-Fi - удобный технический термин Вайб → Настроение, Атмосфера
Гаджет - коротко и понятно Пати → Вечеринка, Праздник
Флешмоб - точное название явления Свэг → Стиль, Харизма
Блог - коротко и понятно Кринж → Стыдно, Неловко
Смартфон - удобно для обозначения типа телефона Мем → Шутка, Прикол, Фраза-шаблон
Дедлайн - в профессиональной среде понятно Флекс → Хвастаться, Показывать

ГЛАВНЫЙ ВЫВОД

Англицизмы — не враги и не друзья. Они просто инструмент.
Используй их разумно: когда это действительно нужно,
но не забывай о богатстве русского языка!
